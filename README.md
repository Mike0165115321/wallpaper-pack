# WangLin Desktop Pack — 望林

Minimalist desktop wallpaper displays with Thai calendar, architectural particle networks, and mouse parallax.

### Wallpapers

| Name | Description |
|------|-------------|
| **WangLin Mountain** | Mountain landscape + glass calendar + floating clock + particle network + mouse parallax |

### Usage

1. Open any `.html` file in a browser
2. Set as full-screen desktop display
3. Move mouse for parallax + shadow effect

### Adding new designs

1. Copy `wanglin-mountain.html` → `wanglin-xxx.html`
2. Change background image, fonts, colors
3. Add entry to `index.html`
4. Commit

### Requirements

- Modern browser (Chrome, Edge, Firefox)
- Sarabun font (auto-loaded from Google Fonts)
- Internet for font + background image loading
